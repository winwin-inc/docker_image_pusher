# 浏览器自动化删除阿里云镜像 Tag 方案

## 方案概述

由于 Docker Registry HTTP API 返回 401 权限不足，可以使用浏览器自动化方式删除阿里云镜像 tag。

## 实现方式

### 方案 1: 使用 Playwright（推荐）

#### 安装
```bash
pip install playwright
playwright install chromium
```

#### 实现思路

1. **登录阿里云控制台**
   - URL: https://cr.console.aliyun.com/
   - 使用账号密码或扫码登录

2. **导航到目标镜像**
   - 选择区域（如北京）
   - 选择命名空间（如 winwin/tool）
   - 找到镜像仓库

3. **删除指定 Tag**
   - 点击镜像仓库
   - 找到指定 tag
   - 点击删除按钮
   - 确认删除

#### 关键挑战

1. **验证码处理**: 阿里云登录可能需要验证码
   - 解决方案: 使用保存的登录状态（cookies）
   - 或使用无头模式时手动处理一次验证码

2. **页面元素定位**: 页面结构可能变化
   - 解决方案: 使用稳定的定位器（data-testid, aria-label 等）
   - 定期更新定位器

3. **批量删除**: 需要逐个删除
   - 解决方案: 循环处理，添加延迟避免触发限制

### 方案 2: 保存登录状态（推荐用于生产）

```python
from playwright.sync_api import sync_playwright
import json

def save_context():
    """保存登录状态"""
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=False)
        context = browser.new_context()

        # 打开页面，手动登录
        page = context.new_page()
        page.goto("https://cr.console.aliyun.com/")

        print("请在浏览器中手动登录...")
        print("登录成功后，按回车键继续...")
        input()

        # 保存 cookies 和 storage state
        context.storage_state(path="aliyun_state.json")
        browser.close()
```

### 方案 3: 使用 MCP Playwright 服务器

您的系统已安装 Playwright MCP 服务器，可以直接使用：

## 实际建议

### 短期方案

1. **手动批量删除**（适合少量 tags）
   - 使用阿里云控制台
   - 支持多选删除

2. **使用浏览器脚本**（适合中等数量）
   - 浏览器控制台 JavaScript
   - 自动化部分操作

### 长期方案

1. **联系阿里云开通 API 权限**
   - 提交工单说明使用场景
   - 申请容器镜像服务的删除 API 权限
   - 可能需要企业版或特定配置

2. **使用阿里云 RAM 授权**
   - 在 RAM 控制台配置权限
   - 为账号添加 `DeleteRepoTag` 权限
   - 重新生成 AccessKey

3. **使用阿里云 CLI**（需要配置权限）
   ```bash
   pip install aliyun-cli
   aliyun configure
   aliyun cr DeleteRepoTag --InstanceId <id> --RepoId <repo_id> --Tag <tag>
   ```

## 浏览器控制台脚本示例

在阿里云控制台按 F12 打开开发者工具，在 Console 中执行：

```javascript
// 批量删除当前页面的 tags
document.querySelectorAll('.delete-button').forEach((btn, index) => {
    setTimeout(() => {
        console.log(`删除第 ${index + 1} 个 tag`);
        btn.click();
        // 确认删除
        setTimeout(() => {
            const confirmBtn = document.querySelector('.confirm-delete');
            if (confirmBtn) confirmBtn.click();
        }, 500);
    }, index * 2000);
});
```

## 总结

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **Docker API** | 快速、可靠 | 需要权限 | 账号有权限时 |
| **浏览器自动化** | 绕过 API 限制 | 依赖页面结构、不稳定 | 临时方案 |
| **手动控制台** | 灵活、可控 | 需要手动操作 | 少量删除 |
| **阿里云 CLI** | 官方支持 | 需要配置权限 | 生产环境 |
| **申请权限** | 一劳永逸 | 需要审核 | 长期方案 |

## 最佳实践建议

1. **优先申请 API 权限**：联系阿里云客服或提交工单
2. **短期使用浏览器脚本**：通过控制台 JavaScript 辅助
3. **长期使用阿里云 CLI**：配置好权限后自动化
